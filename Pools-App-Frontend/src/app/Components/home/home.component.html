Welcome To Pool's App
This is a demo app to test different deployment strategies.

It comes with a front end and a backend.

You can enable or disable backend features like DB connectivity and authentication using the gear icon at the right corner. Once you enable authentication you will need to authenticate yourself before accessing the DB page. When DB connectivity is diabled it will feed you non-persistent static data. Go through the comments for better understanding of the working and how can you modify it to suit your deployment needs. NOTE THIS APP IS FOR TESTING PURPOSES ONLY AND IS NOT MEANT TO BE DEPLOYED FOR PRODUCTION USAGE.
